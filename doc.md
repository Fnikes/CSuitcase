## 下载工具
### axel
```
axel -n 8 -a https://www.xmind.net/xmind/downloads/XMind-ZEN-for-Linux-64bit.deb
```
> -n 指定线程个数 -a 格式化人性输出

## 包管理器
### dpkg 
```
dpkg -i XMind-ZEN-for-Linux-64bit.deb
```
> -i 指定要安装的包


## 常用命令 
### alias 别名
```
alias ls='ls -l'
```
> 使用格式 alias 别名='要执行的命令'
> 用来给程序起别名用的
### pkill
```
pkill 指定进程
```
> 结束指定进程



## 实用工具
### tldr (和man差不多的东西)
用来获取命令最简单直接的使用方式，不用像man那样繁杂
```
" 使用方式
tldr 目标命令
```
